# Documentation for Law Real Estate Website

## Features
This real estate website provides a comprehensive platform for buying, selling, and renting properties. It features:
- **User Registration & Login**: Users can create an account and log in.
- **Property Listings**: Display of available properties with images and descriptions.
- **Search Functionality**: Users can search for properties based on various criteria.
- **Admin Dashboard**: Admins can manage properties and user accounts.

## How to Use
1. **User Registration**: Click on the 'Register' button on the homepage to create an account.
2. **Login**: Enter your credentials (username: `admin`, password: `password123`) to access the dashboard.
3. **Browse Properties**: Use the search bar to find properties. Click on any property to see more details.

## How to Add/Edit/Delete Properties
- **Add Properties**: Admins can add new properties through the dashboard by filling in the property details and clicking the 'Add' button.
- **Edit Properties**: Select an existing property and click 'Edit' to modify details.
- **Delete Properties**: Click the 'Delete' button on the property detail page to remove a property.
